# ♻️ Reloop | Second-Hand Fashion Marketplace

![Reloop Banner](https://images.unsplash.com/photo-1558769132-cb1aea458c5e?ixlib=rb-4.0.3&auto=format&fit=crop&w=1200&q=80)

**Reloop** is a community-driven marketplace designed to modernize the thrifting experience. We provide a seamless, high-end platform for buying and selling second-hand items, focusing on reducing textile waste and promoting a circular economy.

---

## 🌟 Key Features

- **📸 Smart Listing:** Easily upload items with high-quality image support and automated categorization.
- **💎 Curated Collections:** Discover unique vintage finds through personalized feeds and community picks.
- **🔒 Secure Transactions:** Integrated payment protection and secure checkout for buyers and sellers.
- **🌍 Sustainability Tracker:** Visual metrics showing your environmental impact (water saved/CO2 reduced).
- **💬 Direct Messaging:** Real-time chat for negotiations and coordination.

---

## 🛠 Tech Stack

Built with a focus on speed, scalability, and a "pixel-perfect" user experience:

- **Frontend:** Next.js (App Router), React, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Prisma ORM
- **Database:** PostgreSQL
- **Auth:** Clerk / NextAuth
- **Storage:** Cloudinary (for high-res product imagery)
